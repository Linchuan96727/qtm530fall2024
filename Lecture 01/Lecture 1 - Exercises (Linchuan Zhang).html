(f) Generate "vec_x" from a chi-square
distribution with (df = 1, size = 1000)

  • Plot a histogram of each variable, with axis labels, and title
  • Create a new variable "vec_y" by the formula

$ y = e^x + 2x + 5$

  • Plot "vec_x" against "vec_y" and label the axes
In[]:
import numpy as np

np.random.seed(888)

vec_x = np.random.chisquare(df=1, size=1000)

# Create a new variable vec_y
e = np.exp(1)
vec_y = e**vec_x+2*vec_x+5

(g) Research online how to create an array with
size = 1000 entries, with all entries equal to one.

  • Create a new matrix, X_mat that appends a

vector with only ones and "vec_x" from question (f) to form
a matrix with two columns and 1000 rows

  • Compute $(X_{mat}'X_{mat})^{-1}$
In[]:
# Create a new matrix
newarray = np.ones(1000)
X_mat = np.column_stack([newarray,vec_x])

# Calculation
X_mat_transpose = np.matrix.transpose(X_mat)
X_mat_mul=np.matmul(X_mat_transpose,X_mat)
X_inv = np.linalg.inv(X_mat_mul)
print(X_inv)
